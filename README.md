# Chatbot_NVIDIA
Assistente automatizado para atendimento e suporte via chat.

---
## Funcionalidades
* Respostas automatizadas em tempo real.
* Integração com modelo de linguagem para processamento de texto.
* Histórico de conversas simplificado.

---

## Pre-requisitos
* Python 3.10 ou superior
* Token ou chave de API do provedor de IA

---

## Instalacao e Execucao

1. Clone o repositorio:
```
git clone [https://github.com/usuario/projeto.git](https://github.com/usuario/projeto.git)
cd projeto
Instale as dependencias:

Bash
pip install -r requirements.txt
Crie um arquivo .env na raiz do projeto e configure suas credenciais:

Snippet de código
API_KEY=sua_chave_aqui
Inicie o chatbot:

Bash
python app.py
